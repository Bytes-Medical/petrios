# Petrios compliance evidence pack

These documents help an operator assess a specific deployment. They are not a
certification, completed submission, legal advice, or a substitute for evidence
from the production environment.

- [`dtac.md`](./dtac.md) — DTAC evidence workbook and deployment gaps.
- [`dpia-template.md`](./dpia-template.md) — detailed DPIA starter.
- [`../../spec/13-privacy-security-and-compliance.md`](../../spec/13-privacy-security-and-compliance.md)
  — canonical implementation contract for privacy, browser security, public
  disclosures, and accessibility posture.
- [`../../SECURITY.md`](../../SECURITY.md) — vulnerability reporting and
  automated security scanning.

The public runtime pages are `/privacy`, `/privacy/choices`, `/subprocessors`,
and `/data-processing-agreement`. Production is not release-ready until their
operator-specific fields are complete and have been reviewed by the controller.
